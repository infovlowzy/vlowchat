# Welcome to your Vlowchat AI

## Project info

Vlowchat is an OmniChannel Platform built to help businesses with multi-channel AI-based customer service such as WhatsApp, website inquiries, DMs, into a single, all-in-one integrated platform.

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
